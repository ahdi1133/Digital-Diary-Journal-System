# Submission Guide

## Important

This project is a Java Swing desktop application. It should not be deployed on Vercel because Vercel is for web applications, while this project runs as a desktop GUI program.

## What to Submit

Submit the project folder or the ZIP file:

`Digital_Diary_Journal_System_Submission.zip`

## How the Evaluator Can Run It

### Option 1: IntelliJ IDEA

1. Open IntelliJ IDEA.
2. Open the project folder.
3. Let IntelliJ load the `pom.xml` Maven file.
4. Run:

`src/main/java/com/digitaldiary/Main.java`

### Option 2: PowerShell

Run:

```powershell
.\scripts\run.ps1
```

### Option 3: Double-click Launcher

Double-click:

`run.bat`

### Run Tests

```powershell
.\scripts\test.ps1
```
